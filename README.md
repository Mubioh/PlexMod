## Overview

PlexMod is a client mod for Minecraft that adds a handful of QoL tweaks for Mineplex. Open the in-game menu with **K**.

**Supports:**
- Forge 1.8.9
- Fabric 26.1.2

---

## Features

| Feature | Description |
|---|---|
| **Quick Join** | Adds a "Join Mineplex" button directly to the main menu |
| **AutoTaunt** | Automatically `/taunt` at the start of a game |
| **AutoGL** | Sends a custom message at the start of a game |
| **AutoGG** | Sends a custom message at the end of a game |
| **AutoFriend** | Automatically accepts all incoming friend requests |
| **Discord Presence** | Displays your current game on Discord |
| **Cleaner Lobbies** | Removes portal sounds and boss bars in the hub |
| **Chat Cycle** | Cycles between Party, Team, and Private Message chat |
